# Homework 001 - Functions

SICP Python Course | Домашка по первой главе

## Задание

Реализуй три базовые функции и используй их в `main.py`.

| Файл | Функция | Описание |
|------|---------|----------|
| `max.py` | `max(a, b)` | Большее из двух чисел |
| `min.py` | `min(a, b)` | Меньшее из двух чисел |
| `pow.py` | `pow(base, exp)` | base в степени exp (через while!) |
| `main.py` | — | Решить 5 примеров |

## Как сдать

1. Форкни этот репозиторий
2. Реализуй функции в файлах
3. Запусти тесты локально: `pytest tests/ -v`
4. Сделай push — GitHub Actions проверит автоматически

## Ожидаемый вывод main.py

```
12
-3
1024
9
15
```

## Проверка

После push смотри вкладку **Actions** — там будет результат автопроверки.

✅ = все тесты прошли
❌ = есть ошибки, смотри логи
